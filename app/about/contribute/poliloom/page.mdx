To get started:

- [Open the loom](https://loom.everypolitician.org).
- You'll be directed to sign in with Wikimedia â€” there's a button on that page to create an account if you don't yet have one.
- Grant PoliLoom the permission to make edits on your behalf.
- Choose the countries and languages you're confident working in, and start the tutorial.
  - If it's your first time, don't enable Advanced mode at this point.
- Once you're in, you can skip the tutorial if you've done it before.
  - If it's your first time with Advanced mode on, you'll have to re-do the first part of the tutorial to get to the Advanced tutorial.
- Once the tutorial is done, you can start assessing what's been extracted.
