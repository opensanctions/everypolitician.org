# Open Source

EveryPolitician is an open source project. All our code is available on [GitHub](https://github.com/opensanctions).

## Contributing

1. Fork the repository you want to contribute to
2. Create a branch for your changes
3. Submit a pull request with a clear description

## Reporting issues

Found a bug or have a suggestion? [Open an issue on GitHub](https://github.com/opensanctions/opensanctions/issues).
